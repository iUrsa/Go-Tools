# 正态频谱工坊 · 旗舰版

一款纯前端音频频谱可视化与视频生成工具，可直接在浏览器中运行，无需安装任何软件。

中文介绍 | [English Introduction](https://github.com/iUrsa/Go-Tools/blob/main/NormSpect%20Studio/README.md)
## 功能介绍

- 导入音频文件，实时生成动态频谱动画
- 将频谱动画与原始音频合成为完整视频并下载
- 所有处理均在本地浏览器完成，音频不会上传至服务器

### 频谱可视化

- **条状柱形**：经典垂直柱状频谱，随音乐节奏跳动
- **平滑曲线**：连续波形曲线，流畅优雅
- 频谱强度采用正态分布算法向画面中心集中，视觉焦点突出

### 视觉自定义

- **四色渐变**：自由设置4种颜色，自动生成平滑过渡
- **三种填充样式**：实心填充 / 空心边框 / 边框+半透明
- **可调参数**：频谱条数（48~128）、边框宽度、内部透明度、背景透明度
- **特效开关**：峰值点标记、发光边缘


### 视频导出

- **导出格式**：WEBM（含原始音频）
- **可选分辨率**：1080p / 720p / 480p / 方形 1080×1080 / 方形 720×720
- **音频模式**：静默渲染（生成时不外放）/ 实时监听
- 生成过程显示进度条


### 界面特性

- **中文/English 一键切换**：所有文字包括动态提示均可双语显示
- **明亮/暗黑双主题**：默认明亮模式，可手动切换
- **播放进度条**：支持拖动跳转，实时显示当前时间与总时长


### 技术栈

 技术 | 用途 
---|---
 Web Audio API | 音频解码与实时频谱分析 
 Canvas 2D | 自定义频谱绘制（柱状与曲线） 
 MediaRecorder API | 音视频合成与录制 
 正态分布算法 | 频谱能量向中心集中，增强视觉表现力 

## 使用说明

### 第1步：导入音频

1. 点击左侧面板的**导入音频**按钮
1. 选择本地音频文件（支持 mp3、wav、ogg 等常见格式）
1. 加载成功后，下方显示文件名及总时长，右侧预览区显示静态频谱

### 第2步：调整频谱样式

- 频谱形状：在「条状柱形」和「平滑曲线」之间切换
- 四色渐变：点击色块分别设置左到右的四种颜色
- 填充样式：选择实心、空心边框或边框加半透明
- 频谱条数：选择 48、64、96、128 条（推荐 96 条）
- 背景透明度：0 为完全透明，100 为纯黑

### 第3步：添加特效

- 峰值线/圆点：在频谱顶部显示峰值标记
- 发光边缘：为频谱添加发光光晕

### 第4步：实时预览

1. 点击右侧**播放**按钮开始播放音频
1. 频谱将随音乐实时跳动
1. 使用进度条拖动或点击可跳转到任意位置
1. **暂停**和**停止**控制播放状态

### 第五步：生成并导出视频

1. 在左侧选择导出分辨率（建议 720p 用于社交分享）
1. 选择音频输出模式：
   - 静默渲染：生成过程不发出声音
   - 实时监听：生成时可听到音频
1. 点击**生成完整时长视频**按钮
1. 等待进度条走完（时长等于音频全长）
1. 完成后点击**下载视频**保存 WEBM 文件

### 第六步：格式转换（可选）

下载的 WEBM 文件如需导入剪映、Premiere Pro 等软件，可用 ` FFmpeg ` 转换为 MP4：

```bash
ffmpeg -i input.webm -c:v libx264 -c:a aac output.mp4
```
